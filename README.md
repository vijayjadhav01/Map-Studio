# Map Studio

**Map Studio** is a Streamlit-based web application to create **publication-ready choropleth maps of India** from simple CSV or Excel data.

It is designed for **journalists, researchers, students, and data storytellers** who want clean, editorial-quality maps without complex GIS tools.

---

## Features

- Upload CSV / paste Excel or Google Sheets data
- Automatically matches Indian states and UTs
- Editorial map styling with readable labels
- Custom titles, sources, annotations, and credits
- High-resolution image export
- Instagram-ready portrait export (4:5)
- Supports multiple colour palettes
- Handles missing data gracefully

---

## Expected Data Format

Your data should contain **two columns**:

```csv
state,value
Maharashtra,34
Kerala,12
Uttar Pradesh,67
```
